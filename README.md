Aplikasi Pembayaran SPP Sederhana

Aplikasi ini dibuat untuk mengelola pembayaran SPP siswa yang terdiri dari dua
bagian utama: halaman user dan halaman admin. Admin dapat mengelola data siswa
dan pembayaran, sedangkan user dapat melihat status pembayaran tanpa login.
